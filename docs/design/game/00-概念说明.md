# 概念说明

> 本文档定义《天下牌》中所有核心概念。每个概念给出**确切定义**、**适用范围**和**示例**。所有其他设计文档引用的术语以本文为准。

---

## 一、游戏状态维度

### 1.1 气数（Vitality）

| 属性 | 定义 |
|------|------|
| **定义** | 每场战斗中玩家的生命值。气数归零则本场战斗失败。 |
| **范围** | 局内（每场战斗单独计算） |
| **初始值** | 100（可通过局外成长提升） |
| **上限** | 初始有固定上限，可通过局外成长提升 |
| **恢复** | 每场战斗开始前自动回满至当前上限 |
| **扣减** | 被对方出牌命中时，扣除伤害值对应的气数 |
| **扣至归零** | 触发"战败流程"：本场战斗结束，扣除天命 |

**示例**：玩家气数上限 100，被对方打出 15 点伤害 → 气数变为 85。下一场战斗开始回满至 100。

---

### 1.2 天命（Destiny）

| 属性 | 定义 |
|------|------|
| **定义** | 整局游戏的"命数"。类似 Roguelike 的剩余"生命次数"。 |
| **范围** | 局内（整局共享） |
| **初始值** | 100 |
| **上限** | 初始 100，可通过局外成长提升（每次 +10 或 +20） |
| **修正** | 极稀有——仅通过特定事件或局外成长解锁 |
| **归零后果** | 整局游戏**彻底失败**，强制返回标题画面，触发局末结算 |
| **扣减时机** | 每场战斗结束（无论输赢）时扣除 |

**扣减公式（普通战斗）**：
```
天命扣减 = CEIL(敌方剩余气数百分比 ÷ 4)
```

- 敌方剩余气数百分比 = (敌方当前气数 ÷ 敌方气数上限) × 100
- CEIL 表示向上取整

**Boss 战斗扣减**：上述结果 × 2

**示例**：
- 普通战斗，敌方剩余气数 80%：CEIL(80% ÷ 4) = CEIL(20) = **扣 20 天命**
- Boss 战斗，敌方剩余气数 60%：CEIL(60% ÷ 4) × 2 = CEIL(15) × 2 = **扣 30 天命**

---

### 1.3 通宝（Tongbao）

| 属性 | 定义 |
|------|------|
| **定义** | 游戏中流通的货币。 |
| **范围** | 局内（仅当局有效，局末结算利息后清空） |
| **获取** | 战斗胜利奖励、事件奖励 |
| **消费** | 黄金台节点消费 |
| **局末转换** | 局末持有通宝按"每 10 通宝 = 1 利息"转换为局外通宝 |
| **局末清空** | 局内通宝在局末结算后归零，不继承到下一局 |

**示例**：局末持有 127 通宝 → 利息 12 通宝 → 局外账户增加 12 通宝

## 二、卡牌系统

### 2.1 扑克牌

| 属性 | 定义 |
|------|------|
| **定义** | 游戏所用的基础牌。标准 54 张扑克牌。 |
| **构成** | 4 种花色 × 13 个点数 + 小王 + 大王 |
| **花色** | ♠（黑桃）、♣（梅花）、♥（红桃）、♦（方片） |
| **点数** | 3~10=面值, J=11, Q=12, K=13, A=15, 2=20, 小王=25, 大王=30 |
| **角色** | 对局中唯一可以打出的牌型构成要素 |

### 2.2 卜辞牌（Oracle Card）

| 属性 | 定义 |
|------|------|
| **定义** | **永久被动**道具，购买后在整局游戏中生效。 |
| **生效方式** | 购买后自动激活效果，无需手动使用 |
| **作用目标** | 对特定牌型（单张/对子/顺子等）的伤害系数进行加成 |
| **获取途径** | 黄金台购买 |
| **叠加上限** | 同种卜辞牌**不可叠加**（每种最多买一次） |

### 2.3 锦囊牌（Strategy Card）

| 属性 | 定义 |
|------|------|
| **定义** | 一次性效果牌，购买后进入**锦囊区**。 |
| **使用时机** | 出牌阶段（不占用出牌次数，但一回合限用一张） |
| **消耗** | 使用后从锦囊区移除 |
| **获取途径** | 黄金台购买 |
| **锦囊上限** | 锦囊区上限 **3 张** |

### 2.4 事件牌（Event Card）

| 属性 | 定义 |
|------|------|
| **定义** | 强力历史事件牌，触发后产生一次性全局效果。 |
| **触发时机** | 在事件节点或特定人物技能下触发 |
| **释放后** | 效果立即生效，牌被消耗 |
| **概率修正** | 持有特定人物影响该事件出现的概率 |
| **获取途径** | 黄金台购买或事件节点触发 |

### 2.5 印记（Seal）

| 属性 | 定义 |
|------|------|
| **定义** | 单张卡牌上的额外效果标签。 |
| **贴附** | 每张牌**最多 1 枚印记** |
| **生效** | 印记跟随牌移动（手牌→打出→弃牌） |
| **获取途径** | 人物技能效果、卜辞牌附带效果、事件牌奖励（**不可直接购买**） |
| **移除** | 拥有印记的牌被消耗后，印记消失 |

## 三、人物系统

### 3.1 人杰 / 历史人物（Historical Figure）

| 属性 | 定义 |
|------|------|
| **定义** | 玩家招募的文臣武将，提供对局技能加成。 |
| **存在形式** | 队伍列表（不进入手牌，不占用手牌位置） |
| **上阵上限** | 最多同时拥有 **10 个人物** |
| **生效方式** | 人物在队伍中即**全局生效**（整局游戏有效） |
| **获取途径** | 黄金台招募 |
| **失去机制** | 通过事件、部分 Boss 技能、或特定机制失去 |
| **价格** | 按稀有度：金色 120~200、紫色 60~100、蓝色 30~50、白色 10~20、黑色 -30~-10 |

### 3.2 稀有度

| 稀有度 | 颜色 | 价格范围 | 出现概率 | 说明 |
|--------|------|---------|---------|------|
| 负面 | 黑色 | -30 ~ -10（倒贴） | 极低 | 带负面效果，但倒贴通宝 |
| 普通 | 白色 | 10 ~ 20 | 常见 | 基础实用效果 |
| 稀有 | 蓝色 | 30 ~ 50 | 较常见 | 中等强度效果 |
| 史诗 | 紫色 | 60 ~ 100 | 较稀有 | 强力效果 |
| 传说 | 金色 | 120 ~ 200 | 极稀有 | 决定性效果 |

## 四、战斗系统

### 4.1 牌权（Turn / Right to Play）

| 属性 | 定义 |
|------|------|
| **定义** | 当前可以主动出牌的一方。 |
| **初始** | 玩家先手 |
| **转移** | 对方放弃出牌后，造成伤害的一方获得牌权 |
| **效果** | 只有持牌权方可以主动出牌；对方只能接牌或放弃 |

### 4.2 手牌（Hand）

| 属性 | 定义 |
|------|------|
| **定义** | 玩家当前持有的牌。 |
| **初始数量** | 17 张 |
| **来源** | 战斗开始时从牌堆抽取 / 重摸手牌 |
| **上限** | 无硬性上限（但手牌越多不一定越好，因部分技能与手牌数相关） |

### 4.3 牌堆（Deck）

| 属性 | 定义 |
|------|------|
| **定义** | 整局对战中可抽取的牌池。 |
| **构成** | 54 张标准扑克牌 |
| **洗牌** | 每次重摸前自动洗牌 |
| **抽空** | 牌堆抽空后，弃牌堆洗入牌堆重新组成新牌堆 |

### 4.4 弃牌堆（Discard Pile）

| 属性 | 定义 |
|------|------|
| **定义** | 已打出或被弃置的牌的存放区。 |
| **流入** | 每次出牌后，打出的牌进入弃牌堆 |
| **洗入牌堆** | 牌堆抽空时，弃牌堆洗入牌堆 |

### 4.5 伤害公式（Damage Formula）

```
伤害值 = ∑(该手牌中每张牌的面点数) × 牌型系数 × 其他修正
```

| 部分 | 说明 |
|------|------|
| **∑(面点数)** | 该手牌中所有牌的点数之和（按点数对照表） |
| **牌型系数** | 按牌型系数表 |
| **其他修正** | 人物技能、卜辞牌、印记等带来的加成/削减 |

## 五、地图系统

### 5.1 局（Run）

| 属性 | 定义 |
|------|------|
| **定义** | 从天命 100 开始到天命耗尽或通关为止的完整游戏过程。 |
| **结构** | 12 层节点 → 4 个 Boss |
| **结束条件** | 天命归零（失败）或击败第 12 层 Boss（通关） |

### 5.2 层（Floor / Layer）

| 属性 | 定义 |
|------|------|
| **定义** | 地图上的一个节点行。每层包含 1 个节点（Boss 层）或多个可选节点（非 Boss 层）。 |
| **范围** | 共 12 层 |
| **Boss 层** | 第 3、6、9、12 层 |

### 5.3 节点（Node）

| 属性 | 定义 |
|------|------|
| **定义** | 地图上的一个行动点。玩家选择路径后进入节点执行对应内容。 |
| **类型** | 普通战斗、精英战斗、黄金台（商店）、事件、Boss |

---

## 六、游戏阶段

### 6.1 局内（In-Run）

| 属性 | 定义 |
|------|------|
| **定义** | 从选定角色进入地图到天命耗尽/通关的完整过程。 |
| **特点** | 所有资源（通宝、人物、卜辞牌、印记）仅在局内有效 |

### 6.2 局外（Out-of-Run / Meta Progression）

| 属性 | 定义 |
|------|------|
| **定义** | 跨局的持久成长系统。 |
| **内容** | 牌组强化、印记解锁、天命上限提升、初始通宝增加 |
| **消耗** | 局末利息积累的局外通宝 |

## 七、牌型对照

### 7.1 斗地主牌型

| 牌型 | 定义 | 示例 |
|------|------|------|
| **单张** | 任意一张牌 | ♠5 |
| **对子** | 两张同点数牌 | 8-8 |
| **三张** | 三张同点数牌 | K-K-K |
| **三带一** | 三张同点数 + 一张任意单张 | K-K-K + ♠5 |
| **三带二** | 三张同点数 + 一对子 | K-K-K + 7-7 |
| **顺子** | 5 张或以上连续点数的牌 | 3-4-5-6-7 |
| **连对** | 3 对或以上连续点数的对子 | 3-3-4-4-5-5 |
| **飞机** | 2 组或以上连续三张（可带单张或对子） | 3-3-3-4-4-4 |
| **炸弹** | 四张同点数牌 | A-A-A-A |
| **王炸** | 小王 + 大王 | 小王 + 大王 |

### 7.2 牌型比较规则

| 规则 | 说明 |
|------|------|
| 同牌型比较 | 比较**最小牌的点数**，大者胜 |
| 炸弹 | 可管任何非炸弹牌型 |
| 王炸 | 可管任何牌型（包括炸弹） |
| 不同牌型 | 不可互相比较（例如对子不能管单张） |
| A 在顺子中 | A 可作 1（A-2-3-4-5）或作高（10-J-Q-K-A），但不能两个方向同时使用 |

## 八、其他术语

| 术语 | 定义 |
|------|------|
| **放弃** | 无牌可出或选择不出，将牌权拱手让人并接受伤害 |
| **清空手牌** | 某一手出牌正好用完所有手牌，触发清空奖励伤害 ×5 |
| **黄金台** | 地图上的商店节点，消费通宝招募人杰或购买道具 |
| **出牌阶段** | 持有牌权方选择牌型打出的阶段 |
| **接牌阶段** | 非牌权方选择是否用更大牌型回应的阶段 |
| **结算** | 一局结束后的统计算息过程 |
| **重摸** | 清空手牌后从牌堆中重新抽取 17 张手牌 |
| **修正系数** | 所有来自人物技能、卜辞牌、印记的系数调整（加法叠加） |
