## 1.1 游戏概念

《天下牌》是一款以**中国历史文化**为题材的 **Roguelike PVE 卡牌对战游戏**。玩家操控一副标准 54 张扑克牌，穿越历史长河，面对逐步增强的敌人，招募文臣武将，最终战胜 Boss 通关。

**主题特色**：区别于西方卡牌游戏的魔法/奇幻主题，本游戏以中国历史事件、人物、典故为内容载体，扑克牌战斗方式为斗地主牌型规则。

---

## 1.2 核心玩法三句

1. **出牌对战**：使用 54 张扑克牌，按斗地主牌型规则展开对战。双方轮流出牌，拼点数、拼牌型、拼策略，造成伤害耗尽对方气数即获胜。
2. **Roguelike 爬塔**：在 12 层分支地图上行进。每层选择路径，每 3 层面对一个 Boss。节点的选择（战斗/商店/事件）决定构筑方向。
3. **黄金台招募**：用战斗获得的通宝招募历史人物、购买卜辞牌/锦囊牌/事件牌，构筑专属战术体系。人物技能与牌型配合产生协同效应。

---

## 1.3 游戏目标

| 目标层级 | 说明 |
|---------|------|
| **短期目标（每场战斗）** | 出牌耗尽敌方气数，赢得本场战斗 |
| **中期目标（每局游戏）** | 通过 12 层地图、击败 4 个 Boss，完成一次通关 |
| **长期目标（局外成长）** | 积累通宝解锁局外成长，提升初始牌组、天命上限等，挑战更高难度 |

---

## 1.4 核心循环

### 一局游戏的完整流程

```
一局开始（天命 = 100）
  │
  ├─ 第 1-2 层：选择路径 → 进入节点
  │   ├─ 普通战斗 → 获得通宝
  │   ├─ 黄金台 → 消费通宝招募/购买
  │   └─ 事件节点 → 随机奖励或惩罚
  │
  ├─ 第 3 层 → Boss 战
  │
  ├─ 第 4-5 层：同上
  ├─ 第 6 层 → Boss 战
  │
  ├─ 第 7-8 层：同上
  ├─ 第 9 层 → Boss 战
  │
  ├─ 第 10-11 层：同上
  └─ 第 12 层 → 最终 Boss 战
       │
       ├─ 胜利 → 通关 → 结算通宝利息 → 局外成长解锁
       └─ 战败 → 扣天命
              ├─ 天命未耗尽 → 继续下一场
              └─ 天命耗尽 → 游戏结束 → 结算通宝利息
```

### 每场战斗的循环

```
进入战斗 ─→ 双方摸17张手牌 ─→ 玩家先手出牌
    │                              │
    │                         出牌阶段
    │                         (玩家打出一手牌型)
    │                              │
    │                         接牌阶段
    │                         (AI必须用更大同牌型接，或放弃)
    │                              │
    │                     ┌─── 放弃 ───→ 玩家获得牌权
    │                     │              并造成伤害
    │               AI接牌 ──→ 玩家接牌阶段
    │                              │
    │                     ┌─── 玩家放弃 → AI获得牌权
    │                     │              并造成伤害
    │               玩家接牌 → 重复出牌/接牌
    │                              │
    │                   清空手牌？──→ 伤害×5，重新摸17张
    │                              │
    │                   气数归零？──→ 战斗结束
    │                              │
    └──── 对方气数归零 ──→ 战斗胜利
```

---

## 1.5 胜利与失败条件

| 条件 | 结果 |
|------|------|
| 战斗中耗尽敌方气数 | 本场战斗胜利，获得通宝奖励 |
| 战斗中气数归零 | 本场战斗失败，扣除天命 |
| 天命归零 | 整局失败，进入局末结算 |
| 击败第 12 层 Boss | 通关，进入局末结算 |

---

## 1.6 关键设计原则

1. **策略深度**：牌型选择（单张大点 vs 组合轰炸）和路径选择（战斗 vs 商店 vs 事件）都产生有意义决策。
2. **构筑驱动**：卜辞牌、人物技能、印记三者构成协同体系，每局的构筑方向不同。
3. **风险梯度**：普通战斗（低风险低收益）→ 精英战斗（中等风险高收益）→ Boss 战（高风险高收益），玩家可主动选择风险等级。
4. **容错机制**：双生命值系统（气数+天命）允许"战斗失败但仍可继续"，避免一败即死的挫败感。
