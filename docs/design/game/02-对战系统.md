## 2.1 牌组基础

- 使用标准的 **54 张扑克牌**：4 种花色（♠♣♥♦）× 13 个点数 + 小王 + 大王
- **开局**：双方各摸 **17 张手牌**（从 54 张中随机抽取）
- **先手**：玩家先手出牌
- **牌堆管理**：每次重摸前自动洗牌；牌堆抽空时，弃牌堆洗入牌堆重新组成新牌堆
- **锦囊牌**：锦囊牌在购买后进入锦囊区，锦囊区上限 **3 张**

## 2.3 对局流程（完整版）

### 2.3.1 阶段一：出牌阶段

当前持有牌权的一方执行：

1. 从手牌中选择符合斗地主牌型规则的组合（见 2.5）
2. 点击确认出牌
3. 对方进入接牌阶段

**如果持有牌权方无法出任何合法牌型**：视为自动放弃，牌权转移给对方，不受伤害（因无人造成伤害）。但若对方也已经无牌可出，双方比对剩余气数决定胜负。

### 2.3.2 阶段二：接牌阶段

非牌权方执行：

1. 必须使用**同牌型且点数更大**的组合来回应（比较规则见 2.5.2）
2. 如果有合法接牌，可以选择**接牌**或**放弃**
3. 如果无合法接牌，只能选择**放弃**

### 2.3.3 阶段三：放弃与伤害结算

1. 选择放弃的一方受到对方**最后一手有效出牌的伤害**
2. 伤害公式见 2.4
3. 造成伤害的一方获得牌权，进入新的出牌阶段

### 2.3.4 特殊规则：清空手牌

### 2.3.4 特殊规则：清空手牌

1. **清空判定**：若某一手出牌正好用完所有手牌，称为**清空手牌**。
2. **伤害加成**：本手伤害 **×5**。
3. **结算后摸满**：五倍伤害结算完成后，清空方立即重新摸 **17 张手牌**。
4. **牌权转移**：清空方摸满后，**牌权转移给对方**，由对方开始新一轮出牌（不再限制牌型，自由出牌）。

> （弃置最后一张牌——来自黄天、猛攻等弃置类技能——仅减少手牌，不再立即摸满，其重摸依下方 2.3.6 规则处理。）
> （另一方在清空事件中不摸牌。）

### 2.3.5 获得牌权时的补牌

- 任意一方在**获得牌权**（成为新一轮出牌方）时，若其手牌为空，立即摸满 **17 张手牌**。
- 此规则作为安全网，确保任何因弃置类技能清空手牌的一方在拿到牌权时不会被卡在 0 牌状态。

> **示例**：玩家手牌只剩一张大王，出单张大王 → 清空手牌 → 大王伤害 (30×1)×5 = **150 伤害** → 玩家摸满 17 张 → 牌权转移给对方，对方自由出牌。

### 2.3.6 胜负判定

- **气数归零**：一方气数归零，本场战斗立即结束
- **宽平和解**：双方连续均放弃（无人可出合法牌型）时，比较剩余气数百分比
- **战斗胜利**：对方气数归零，己方存活 → 战斗胜利

## 2.4 伤害公式（正式版）

```
伤害值 = [∑(每张牌的面点数) × 牌型系数] × 清空手牌倍数 + 固定加成
```

### 2.4.1 牌型系数表

| 牌型 | 系数 | 说明 | 示例 |
|------|:----:|------|------|
| 单张 | ×1 | 任意 1 张 | ♠5 → 5 伤害 |
| 对子 | ×1.2 | 同点 2 张 | ♦7♣7 → (7+7)×1.2 = 17 伤害 |
| 三张 | ×1.5 | 同点 3 张 | ♥K♠K♦K → (13+13+13)×1.5 = 59 伤害 |
| 三带一 | ×1.5 | 三张 + 1 单张 | KKK+5 → (13×3+5)×1.5 = 66 伤害 |
| 三带二 | ×2 | 三张 + 1 对子 | KKK+77 → (13×3+7+7)×2 = 106 伤害 |
| 顺子 | 2 + (n-5)×0.5 | n = 张数，5 张起 | 3-4-5-6-7(5张) → (3+4+5+6+7)×2 = 50 |
| 连对 | ×2 | 3 对起连续对子 | 33-44-55 → (3+3+4+4+5+5)×2 = 48 |
| 飞机 | ×2.5 | 2 组起连续三张 | 333-444 → (3×3+4×3)×2.5 = 53 |
| 四张炸弹 | ×3 | 同点 4 张 | AAAA → (15×4)×3 = 180 |
| 王炸 | ×4 | 小王 + 大王 | (25+30)×4 = 220 |

### 2.4.2 伤害计算示例

| 示例 | 计算过程 | 结果 |
|------|---------|:----:|
| 单张 A | 15 × 1 | **15 伤害** |
| 顺子 3-4-5-6-7-8（6 张，点数 33） | 33 × (2+0.5) | **83 伤害** |
| 炸弹 4 个 K（点数 52） | 52 × 3 | **156 伤害** |
| 王炸（点数 55） | 55 × 4 | **220 伤害** |
| 清空手牌王炸 | 220 × 5 | **1100 伤害**（秒杀级） |
| 对子 88（点数 16） | 16 × 1.2 | **19 伤害** |

### 2.4.3 修正系数叠加规则

所有外部修正（人物技能、卜辞牌、印记）统一以**加法叠加**的方式累加至牌型系数：

```
最终系数 = 原始牌型系数 + 卜辞牌修正 + 人物技能修正 + 印记修正
```

> **示例**：玩家持有"破军·单张"（+0.5 单张系数），同时关羽"武圣"技能生效（单张系数 +0.3），打出单张 A：15 × (1 + 0.5 + 0.3) = 15 × 1.8 = **27 伤害**

## 2.5 斗地主完整牌型规则

### 2.5.1 牌型定义

| 牌型 | 定义 | 最小条件 | 示例 |
|------|------|---------|------|
| 单张 | 任意一张牌 | 1 张 | ♠5 |
| 对子 | 两张同点数牌 | 2 张 | 8-8 |
| 三张 | 三张同点数牌 | 3 张 | K-K-K |
| 三带一 | 三张同点 + 一张任意 | 4 张 | K-K-K + ♠5 |
| 三带二 | 三张同点 + 一对子 | 5 张 | K-K-K + 7-7 |
| 顺子 | 连续点数的牌 | 5 张（无上限） | 3-4-5-6-7 |
| 连对 | 连续点数的对子 | 3 对（6 张） | 3-3-4-4-5-5 |
| 飞机 | 连续点数的三张 | 2 组（6 张） | 3-3-3-4-4-4 |
| 飞机带单 | 飞机 + 相同数量单张 | 飞机组数 + 单张数 | 333-444 + 5 + 6 |
| 飞机带对 | 飞机 + 相同数量对子 | 飞机组数 + 对子数 | 333-444 + 55 + 66 |
| 炸弹 | 四张同点牌 | 4 张 | A-A-A-A |
| 王炸 | 小王 + 大王 | 2 张 | 小王 + 大王 |

### 2.5.2 比较规则

| 规则 | 具体说明 |
|------|---------|
| **同牌型比较** | 比较该手牌中**最小一张牌的点数**，点数大者胜 |
| **跨牌型比较** | 不同牌型**不可互相比较**（对子不能管单张，顺子不能管对子） |
| **特殊：炸弹** | 炸弹可以管任何非炸弹牌型 |
| **特殊：王炸** | 王炸可以管任何牌型（包括炸弹） |
| **顺子长度** | 比较顺子时，**张数必须相同**才能接（5 张顺只能被 5 张顺接） |
| **连对长度** | 比较连对时，**对数必须相同**（3 连连对只能被 3 连连对接） |
| **飞机长度** | 比较飞机时，**组数必须相同**（2 组飞机只能被 2 组飞机接） |

### 2.5.3 A 在顺子中的规则

A 在顺子中有两种合法用法，**但同一手顺子中只能选择一种**：

1. **A 作 1**：A-2-3-4-5（这是唯一包含 2 的顺子组合，A 在此视为 1）
2. **A 作高**：10-J-Q-K-A（A 在此视为 15，管 9-10-J-Q-K）

> A-2-3-4-5 中 2 参与顺子是**唯一例外**。其他任何顺子**不可包含 2**或大小王。

### 2.5.4 合法牌型判定规则

- 顺子不可包含 2、小王、大王（A-2-3-4-5 除外，此情况下 A 视为 1）
- 连对不可包含 2、小王、大王
- 飞机不可包含 2、小王、大王
- 炸弹由四张同点数牌组成，不区分花色（可以是四种花色混合）
- 三带一/三带二中的"三张"不可与"带"的部分点数相同（不能 333+3）

## 2.6 AI 出牌逻辑概要

AI 对手的出牌遵循以下优先级（详细实现见开发文档）：

1. **最小牌优先**：AI 倾向于用能出的最小牌型接牌（模拟人类新手策略）
2. **放弃条件**：从低到高遍历所有可接牌型，若均大于当前牌型所需点数，选择放弃
3. **特殊 AI 行为**：Boss 可按 Boss 技能规则偏离上述默认逻辑

## 2.7 数值平衡要点

| 要点 | 说明 |
|------|------|
| 大小王的强度 | 王炸固定 220 伤害，配合清空手牌可达 1100（近乎秒杀），应控制小王/大王的获取频率 |
| 炸弹的稀有度 | 4 张同点牌在 17 张手牌中概率较低，因此 ×3 系数合理 |
| 顺子的掣肘 | 顺子需要连续点数，构筑成本高但系数增长快（每多一张 +0.5），是长线构筑方向 |
| 通宝经济平衡 | 一场战斗获得 8-15 通宝，金色人物 120-200，需要 8-25 场胜利才够买，控制通关节奏 |
