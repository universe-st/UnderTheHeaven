## 4.1 系统总览

本游戏采用**双生命值系统**——"气数"和"天命"——分别对应**微观（每场战斗）**和**宏观（整局游戏）**两个层次的生存状态。

| 维度 | 气数（Vitality） | 天命（Destiny） |
|:----:|:-----------------:|:----------------:|
| 范围 | 局内·每场战斗独立 | 局内·整局共享 |
| 类比 Roguelike | 单场战斗的 HP | 整局的"命数/生命" |
| 归零后果 | 本场战斗失败，扣除天命 | 整局游戏彻底失败 |
| 恢复方式 | 每场战斗开始前自动回满 | 极稀有（特定事件/局外成长） |

---

## 4.2 气数（战斗中生命值）

### 4.2.1 基本规则

| 属性 | 数值 |
|------|:----:|
| 初始上限 | 100 |
| 当前值 | 每场战斗开始前**自动回满**至当前上限 |
| 上限提升 | 通过局外成长系统可提升（具体提升值见 10-局外成长系统.md） |
| 扣减 | 被对方出牌命中时，扣除伤害值对应的气数 |

### 4.2.2 气数变化时机

| 时机 | 变化 | 说明 |
|:----:|:----:|------|
| 战斗开始 | 恢复至上限 | 无论上一场战斗结束时气数多少，新战斗开始回满 |
| 被出牌命中 | 减少 | 减少量 = 对方出牌的伤害值 |
| 人物技能恢复 | 增加 | 如华佗"青囊"、刘秀"光武"等技能在满足条件时恢复 |
| 气数归零 | 战斗结束 | 触发天命扣减流程 |

### 4.2.3 气数归零流程

```
气数 ≤ 0
  │
  ├─ 本场战斗立即结束，判定为失败
  ├─ 不结算通宝奖励
  └─ 进入天命扣减流程（见 4.3.3）
```

## 4.3 天命（局外生命值）

### 4.3.1 基本规则

| 属性 | 数值 |
|------|:----:|
| 初始值 | 100 |
| 上限 | 初始 100，可通过局外成长提升（每次 +10 或 +20） |
| 恢复 | 极稀有——仅通过特定事件（如"负荆请罪"不扣天命）或局外成长解锁 |
| 归零后果 | 整局游戏**彻底失败**，回到标题画面，触发局末结算 |

### 4.3.2 天命扣减时机

天命**只在战斗结束后**扣减，不因其他行为扣减。每场战斗（无论胜负）都会触发扣减，除非有特殊效果豁免。

### 4.3.3 天命扣减公式

**标准公式**：
```
天命扣减 = CEIL(敌方剩余气数百分比 ÷ 4)
```

其中：
```
敌方剩余气数百分比 = (敌方当前气数 ÷ 敌方气数上限) × 100
```

**Boss 战斗**：
```
天命扣减 = CEIL(敌方剩余气数百分比 ÷ 4) × 2
```

**取整规则**：CEIL 表示向上取整（如 20.1 → 21）。

### 4.3.4 天命扣减示例

| 战斗类型 | 敌方剩余气数 | 计算过程 | 扣减天命 |
|:-------:|:----------:|:--------:|:-------:|
| 普通 | 80% | CEIL(80 ÷ 4) = CEIL(20) | **20** |
| 普通 | 55% | CEIL(55 ÷ 4) = CEIL(13.75) | **14** |
| 普通 | 25% | CEIL(25 ÷ 4) = CEIL(6.25) | **7** |
| 普通 | 1% | CEIL(1 ÷ 4) = CEIL(0.25) | **1** |
| Boss | 80% | CEIL(80 ÷ 4) × 2 = 20 × 2 | **40** |
| Boss | 55% | CEIL(55 ÷ 4) × 2 = 14 × 2 | **28** |
| Boss | 1% | CEIL(1 ÷ 4) × 2 = 1 × 2 | **2** |

### 4.3.5 敌方气数越少越好

从公式可见：**敌方剩余气数越低，扣除天命越少**。这意味着：

- 虽然战斗失败，但**打得越接近胜利**（敌方血越少），扣除天命越少
- 这就是"虽败犹荣"机制——尽力消耗敌人会有回报
- 被秒杀（敌方满血）是最坏情况

**极端示例**：
- 打出 1100 伤害把 Boss 从 1500 打到 400（剩余 26.7%）：扣 CEIL(26.7÷4) = **7 天命**
- 被 Boss 秒杀（敌方满血 100%）：扣 CEIL(100÷4) = **25 天命**

## 4.4 双生命值系统设计意图

1. **容错空间**：单场战斗失败 ≠ 游戏结束。天命提供了"卷土重来"的机会，降低挫败感。
2. **风险累积**：连续失败导致天命逐渐耗尽，形成渐进式压力而非一刀切的 Game Over。
3. **奖励进取**：战斗失败时扣减的天命与消耗敌人程度挂钩，鼓励玩家即使劣势也要尽力输出。
4. **长线规划**：天命上限提升是重要局外成长目标，为反复游玩提供动力。
