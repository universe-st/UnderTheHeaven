## 5.1 系统定位

战间系统是指**战斗结束后、新战斗开始前**在"层"之间发生的一切操作，主要包括：

1. **通宝获得**：战斗胜利后获得通宝奖励
2. **黄金台消费**：在黄金台节点消费通宝
3. **路径选择**：在层间选择下一节点的路径

---

## 5.2 通宝经济

### 5.2.1 通宝获取方式

| 方式 | 获取量 | 频率 |
|:----:|:------:|:----:|
| 普通战斗胜利 | 8-15 通宝 | 常见 |
| 精英战斗胜利 | 20-30 通宝 | 较常见 |
| Boss 战斗胜利 | 40-60 通宝 | 每 3 层一次 |
| 事件奖励 | 按事件类型 | 按事件出现概率 |
| 人物技能加成 | 朱元璋（+30%）、李善长（+10）等 | 持有即生效 |

### 5.2.2 通宝消费方式

| 方式 | 说明 |
|:----:|------|
| 招募历史人物 | 按稀有度定价 10-200 通宝（见 5.3） |
| 购买卜辞牌 | 30-80 通宝（见 06-牌与道具系统.md） |
| 购买锦囊牌 | 20-50 通宝（见 06-牌与道具系统.md） |
| 购买事件牌 | 50-100 通宝（见 06-牌与道具系统.md） |

### 5.2.3 通宝流动模型

```
战斗胜利 → 获得通宝 → 黄金台消费 → 构筑变强 → 更强战斗
                              ↑                     │
                              └── 更多通宝 ←────────┘
```

## 5.3 黄金台（商店）

黄金台是地图上的重要非战斗节点，模拟古代招贤纳士的"黄金台"典故。

### 5.3.1 黄金台基本规则

| 规则 | 说明 |
|:----:|------|
| 出现位置 | 地图上的黄金台节点（每 4 层内通常至少出现一次） |
| 刷新机制 | 每次进入黄金台，随机刷新**4 件商品**供购买 |
| 商品类型 | 人物招募、卜辞牌、锦囊牌、事件牌 |
| 刷新锁定 | 离开黄金台后，本次刷新的商品不会被保留 |
| 特价商品 | 极低概率出现"免费"商品（受人物技能影响） |

### 5.3.2 商品类型详情

#### A. 历史人物招募

| 稀有度 | 颜色 | 价格（通宝） | 出现概率 | 效果强度 |
|:------:|:----:|:----------:|:--------:|:--------:|
| 负面 | 黑色 | -30 ~ -10（倒贴） | 5% | 负面效果 |
| 普通 | 白色 | 10 ~ 20 | 35% | 基础效果 |
| 稀有 | 蓝色 | 30 ~ 50 | 30% | 中等效果 |
| 史诗 | 紫色 | 60 ~ 100 | 20% | 强力效果 |
| 传说 | 金色 | 120 ~ 200 | 10% | 决定性效果 |

**招募规则**：
- 每次黄金台从所有稀有度池中按出现概率抽取 2-4 个人物
- 上阵上限：最多同时拥有 **10 个人物**
- 如果已达上限，黄金台不再显示人物招募选项
- 持有管仲（通货）→ 商品价格 -15%
- 持有鲁班（巧匠）→ 商品价格 -10%（与管仲叠加，总减免 25%）
- 持有袁绍（四世三公）→ 人物价格 -10 通宝（最低不减至 10 以下）

#### B. 卜辞牌

| 名称 | 效果 | 价格 |
|:----:|:----:|:----:|
| 破军·单张 | 单张伤害系数 +0.5 | 30 |
| 掠地·对子 | 对子伤害系数 +0.3 | 35 |
| 连珠·顺子 | 顺子每张额外 +0.3 系数 | 50 |
| 双飞·连对 | 连对伤害系数 +1 | 45 |
| 冲天·飞机 | 飞机伤害系数 +1 | 55 |
| 天威·炸弹 | 炸弹伤害 +50% | 60 |
| 王权·王炸 | 王炸伤害翻倍 | 80 |

#### C. 锦囊牌

| 名称 | 效果 | 价格 |
|:----:|:----:|:----:|
| 无中生有 | 从牌堆摸 3 张牌 | 30 |
| 釜底抽薪 | 对方随机弃 2 张手牌 | 35 |
| 借刀杀人 | 强制对方出牌；若放弃，则受最大单牌点数 × 系数伤害 | 40 |
| 偷梁换柱 | 将手中 1 张牌与牌堆顶 1 张交换 | 25 |

| 走为上计 | 立即结束本回合，不造成伤害，牌权不转移 | 20 |

#### D. 事件牌

| 名称 | 效果 | 价格 | 提升概率人物 | 降低概率人物 |
|:----:|:----:|:----:|:----------:|:----------:|
| 赤壁之战 | 本回合所有伤害 +50% | 60 | 周瑜、黄盖 | 曹操 |
| 负荆请罪 | 回复 30% 气数 | 50 | 廉颇、蔺相如 | — |

| 卧薪尝胆 | 下一场战斗伤害 +30% | 65 | 勾践 | 夫差 |

### 5.3.3 黄金台交互流程

```
进入黄金台节点
  │
  ├─ 展示 4 件商品（含类型、名称、效果、价格）
  │
  ├─ 玩家选择：
  │   ├─ 购买某商品（消耗对应通宝）
  │   ├─ 查看人物技能详情（如果商品为人物）
  │   └─ 离开黄金台
  │
  ├─ 购买后，商品从列表中移除
  │
  └─ 选择"离开" → 回到地图
```

## 5.4 战间流程（完整版）

### 完整一局的战间流程

```
战斗胜利
  │
  ├─ 获得通宝奖励（基础值 + 人物加成值）
  ├─ 如有"卧薪尝胆"等持续效果，触发效果
  │
  ├─ 回到地图 → 展示当前层可选的下一层路径
  │
  ├─ 玩家选择路径 → 进入节点
  │
  │   ├─ 黄金台：进入 5.3.3 黄金台交互流程
  │   ├─ 事件：触发事件，按事件内容执行
  │   └─ 战斗/精英/Boss：进入战斗
  │
  └─ 进入战斗 → 回满气数 → 摸 17 张手牌 → 战斗开始
```

### 战斗失败流程

```
战斗失败（气数归零）
  │
  ├─ 不获得通宝奖励
  ├─ 计算天命扣减（按 4.3.3 公式）
  ├─ 扣除天命
  │
  ├─ 天命 > 0：回到地图 → 继续下一层
  └─ 天命 ≤ 0：整局结束 → 局末结算
```
