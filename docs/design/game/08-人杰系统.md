# 八、角色系统

## 8.1 系统概述

角色系统提供对战中可选择的玩家角色。每个角色拥有独特的技能，在整局对战中持续生效。

### 基本规则

| 规则 | 说明 |
|:----:|------|
| 角色选择 | 每局游戏开始前，从已解锁的角色中选择一个出战 |
| 费用 | 每个角色有费用值，费用高的角色技能更强 |
| 生效方式 | 角色技能在整场对战中持续生效 |

## 8.2 玩家角色列表

### 韩信（费用 8）

**点兵**：伤害结算时，系数乘以打出牌的花色数。

即：如果打出的牌包含 ♠♣♥♦ 中的 x 种花色，则最终伤害系数乘以 x。

### 刘伯温（费用 5）

**筹算**：你可以将黑桃牌当做癞子牌打出。

即：黑桃牌（♠）可以作为任意花色的牌参与牌型组合。

### 李时珍（费用 3）

**本草**：打出牌时，回复等同于本次打出的所有梅花牌分数之和的气数。

即：打出牌结算时，先统计其中所有梅花牌（♣）的 rank 之和，回复等量气数，然后再进行伤害结算。

### 诸葛亮（费用 8）

**天算**：敌方始终随机有六张牌明置。
**还治**：你可以用相同的牌接住敌方的牌型，而不必用更大的。

天算效果：敌方有 6 张手牌始终正面朝上（明牌），玩家可以看到。每回合开始或敌方摸牌时重新随机选择 6 张。
还治效果：当敌方打出牌型后，如果你手中有完全相同的牌型（同点数、同张数、同花色组合），可以用它来接，不需要比敌方的大。

### 文天祥（费用 5）

**丹心**：你的红桃牌结算伤害 +10。

即：每次打出的牌中包含红桃牌（♥）时，额外造成 10 点伤害。

### 牛皋（费用 3）

**猛攻**：你造成伤害后，若对方手牌数不小于 10，随机弃置其一张牌。

即：每次造成伤害后，检查敌方手牌数，如果 ≥10，随机弃掉敌方一张手牌。

## 8.3 角色构筑建议

在游戏初期，建议优先解锁费用 3 的角色（李时珍、牛皋），逐步挑战高费用角色。

| 角色 | 玩法风格 |
|:----:|:--------:|
| 韩信 | 高伤害输出，需要多种花色配合 |
| 刘伯温 | 灵活多变，癞子牌可弥补牌型缺陷 |
| 李时珍 | 续航回复，适合持久战 |
| 诸葛亮 | 信息优势 + 防守反击 |
| 文天祥 | 爆发型红桃输出 |
| 牛皋 | 压制型，削减敌方手牌 |
