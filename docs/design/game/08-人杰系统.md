## 8.1 系统概述

人杰系统表示玩家招募的历史文臣武将。每个人物提供独特技能效果，在整局游戏中持续生效。人物是构筑策略的核心——不同人物的组合决定了玩法的走向。

### 基本规则

| 规则 | 说明 |
|:----:|------|
| 上阵上限 | 最多同时拥有 **10 个人物** |
| 生效方式 | 人物在队伍中即**全局生效**，无需手动激活 |
| 获取途径 | 黄金台招募 |
| 失去途径 | 事件、Boss 技能可以导致人物被移除 |
| 稀有度 | 金色（传说）> 紫色（史诗）> 蓝色（稀有）> 白色（普通）> 黑色（负面） |

## 8.3 人物协同构筑建议

| 核心思路 | 推荐人物组合 | 说明 |
|:--------:|:-----------:|------|
| 顺子流 | 霍去病 + 周瑜 + 韩信 + 李白 | 霍去病减少顺子长度，周瑜增加伤害，韩信用兵仙压制，李白使连对/顺子互通 |
| 炸弹流 | 姬发 + 铁木真 + 魏征 | 姬发和铁木真同时加炸弹系数，魏征在对方炸弹时摸牌 |
| 单张大牌流 | 关羽 + 李广 + 薛仁贵 + 黄忠 | 关羽和薛仁贵增强单张，李广加点数，黄忠改变系数 |
| 防守续航流 | 李世民 + 墨翟 + 华佗 + 包拯 | 李世民减首伤，墨翟放弃回血，华佗开局回血，包拯防负面 |
| 经济流 | 朱元璋 + 商鞅 + 李善长 + 管仲 | 通宝获取最大化 + 黄金台打折 |
