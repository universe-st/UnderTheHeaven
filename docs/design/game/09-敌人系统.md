# 九、敌人系统

## 9.1 系统概述

敌方角色是玩家在对战中遭遇的对手。每个敌方角色拥有独特技能，改变对局规则。

## 9.2 敌方角色列表

### 黄巾军

**黄天**：获得牌权时，随机弃置一张点数最小的牌并摸一张。

即：每当黄巾军获得出牌权（主动出牌或玩家不出时），从手牌中随机选择一张点数最小的牌弃掉，然后摸一张牌。

### 南蛮军

**藤甲**：敌方的黑色牌不计算伤害，红桃牌结算伤害乘以 2。

即：当南蛮军受到伤害时，如果打出的是黑色牌（♠♣），该牌不计算伤害；如果打出的是红桃牌（♥），伤害乘以 2。其他牌（♦）正常计算。

### 强盗

**剪径**：造成伤害后，随机获得你的一张牌。

即：每次强盗造成伤害后，从玩家手牌中随机获取一张牌（加入强盗手牌）。
