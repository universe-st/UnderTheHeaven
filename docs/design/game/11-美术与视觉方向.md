# 十一、美术与视觉方向

> 对应主策划案章节：[主策划案](./主策划案.md)

---

## 11.1 整体风格

- **中国水墨风 + UI 卡片质感**（黑白为主，彩色点缀稀有度）
- 扑克牌采用中国古风纹样（云纹、水纹、龙纹）替代传统花色
- 人物卡牌使用水墨肖像画风格

---

## 11.2 牌面设计

| 区域 | 设计说明 |
|:----:|---------|
| 点数区 | 使用中文数字（三四五六七八九十 JQKA）或篆书变体 |
| 花色区 | ♠♣♥♦ 保留，但背景加入中国纹样（云纹、水纹、龙纹） |
| 人物卡 | 人物剪影 + 姓名 + 技能简述 |
| 稀有度标识 | 金色/紫色/蓝色/白色/黑色 边框或背景色 |
| 印记标记 | 印记以小图标形式显示在牌的右下角 |

---

## 11.3 牌面样式

### 扑克牌样式

```
┌─────────┐
│ ♠       │  ← 花色标记（左上角）
│   三     │  ← 点数（中文数字）
│         │
│  ～～～  │  ← 背景纹样（云纹）
│         │
│        ♠│  ← 花色标记（右下角）
└─────────┘
```

### 人物卡样式

```
┌─────────┐
│ 金色边框 │  ← 稀有度边框
│         │
│  /\/\/\ │  ← 人物水墨剪影
│  /    \ │
│ 诸葛亮  │  ← 人物姓名
│ 神机    │  ← 技能名
│ +2点数  │  ← 技能效果简述
└─────────┘
```

---

## 11.4 界面氛围

| 界面 | 设计描述 |
|:----:|---------|
| 背景 | 使用古画长卷风格（《千里江山图》《清明上河图》风格化） |
| 黄金台 | 以古代高台建筑呈现，带有汉白玉栏杆和升腾的云气 |
| 战斗场景 | 简笔山水 + 动态云雾粒子，双方牌桌居中 |
| 地图 | 水墨卷轴展开形式，节点以印章/玉玺样式呈现 |
| UI 控件 | 仿竹简/宣纸质感，按钮用篆刻印章样式 |

## 11.6 动画与动效

| 对象 | 动效 |
|:----:|------|
| 出牌 | 牌飞出动画，带墨迹拖尾效果 |
| 伤害 | 水墨爆裂效果，数字飘出 |
| 清空手牌 | 金色闪光 + 全屏震动 |
| Boss 登场 | 乌云翻滚效果，Boss 剪影放大 |
| 黄金台 | 金币洒落粒子效果 |
